**Overview**
The Student Result Analysis project is a Python‑based data analysis exercise built using NumPy. It demonstrates how numerical computing libraries can be applied to educational datasets to calculate and evaluate student performance across multiple subjects.

**Key Features**
- Data Handling: Reads student marks from structured arrays and exports results into CSV format.
- Computation: Automatically calculates Total Marks and Average Marks for each student using NumPy operations.
- Formatting: Saves results with clear headers and properly aligned columns for easy readability.
- Scalability: Can be extended to include more subjects, grading logic, or statistical insights.

  **Learning Objectives**
- Practice NumPy array manipulation and mathematical operations.
- Understand how to structure tabular data for analysis.
- Gain experience in exporting results to CSV for reporting and visualization.
  
**Use Cases**
- Teachers or institutions can quickly generate performance summaries.
- Students learning Python/NumPy can use this project as a hands‑on example of applying theory to real‑world data.
- Serves as a foundation for more advanced analytics, such as ranking, grade distribution, or visualization with Pandas/Matplotlib.
